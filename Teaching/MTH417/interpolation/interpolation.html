Lagrange Interpolation

  • Select some interpolation points (x_0,y_0), (x_1,y_1), (x_2,y_2),... by clicking on the canevat below. The inteprolation polynomial is drawn interactively.
  • To select points associated to a continuous functions f=y(x), select a continuous function (1 : f(x)=sin(x), 2 : f(x)=exp(-1/x*x), 3 : f(x) piecewise linear).
