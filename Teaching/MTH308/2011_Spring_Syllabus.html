    • Minimal Outcome. This course is to provide students with quantitative and problem-solving skills for first order, linear second order and systems of linear differential equations. At the conclusion of this course, students should be able to:

    • Understand the concepts of solution, initial value problem and direction field.
    • Solve first order linear differential equations.
    • Solve second order homogeneous and non-homogeneous linear ODEs.
    • Use elimination method to solve systems with constant coefficients.
    • Use matrix methods to solve linear systems of ODEs/
    • Use Laplace transform methods to solve linear systems of ODEs.
    • Understand and apply basic numerical methods for solving.
