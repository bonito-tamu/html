Course description and prerequisites

This is a one-semester course on numerical analysis which gives an introduction to various topics in numerical methods and provides a basis for future study. As well as lectures, there will be homework and programming assignments designed to provide concrete illustration of the proposed algorithms. It is suggested that the programming assignments be completed using MATLAB although other computing languages such as C, C++ or FORTRAN can also be used.

Prerequisites: Basic linear algebra and a course on ordinary differential equations. Familiarity with MATLAB programming or other modern computing language is also required.

Minimal Outcome

Interpolation, numerical evaluation of definite integrals and solution of ordinary differential equations; stability and convergence of methods and error estimates.
