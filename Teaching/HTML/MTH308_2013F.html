Useful Matlab Commands

meshgrid Rectangular grid in 2-D and 3-D space
quiver Quiver or velocity plot
ones Create array of all one
Example to plot director fields for y'=exp(-t)-2y [T,Y] = meshgrid(-2:0.2:3,-1:0.2:2); S=exp(-T)-2*Y; quiver(T,Y,ones(size(S)),S), axis tight
plot 2D line plot
Example using plot T =(0:0.1:2); Y=sin(T); plot(T,Y)
ode45 ODE solver
Example using ode45 [T,Y]=ode45(@(t,y) exp(-t)-2*y,-1:0.2:2,0)
